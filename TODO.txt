getXXX use of indexes:
1) check if there is a unique index that contains all the where-properties. if so use that to return the matching element (if any).
2) check all indexes that contains at least one of the where-properties in it then filter linearly on the shortest index-lookup result.
3) same as 2) but using the <empty> index / all_set (-> "full table scan").
